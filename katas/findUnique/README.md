# Find unique

### Description
There is an array with some numbers. All numbers are equal except for one. Try to find it!

### Examples
```typescript
findUnique([ 1, 1, 1, 2, 1, 1 ]) // returns 2
findUnique([ 0, 0, 0.55, 0, 0 ]) // returns 0.55
```
